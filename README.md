# habittracker
## Project Description
This is a Python-based habit tracking application. It allows users to create, delete, complete, and analyse habits using a command-line interface.

##Features 
-it includes 5 predefined habits
-create habits
-delete habits
-check off habits
-view all habits
-store habit in SQLite file
-analyse habit streaks
-filter habits by periodicity

##Technologies used
-Python 3.7+
-SQlite storage file
-Command Line Interface
-pytest for testing
